# Simulado B1 — Desenvolvimento em JavaScript

Site interativo para estudar a prova do **Professor Argentino**, criado para a turma de **Análise e Desenvolvimento de Sistemas — 3º semestre**.

Acesse em: [Landing Page](https://jandersonhp.github.io/simuladob1/)

> **Como este projeto foi feito:** Este site foi criado inteiramente com **OpenClaude** usado diretamente pelo terminal. Os arquivos dos exercícios das aulas foram lidos pela IA, que gerou automaticamente os simulados interativos, o CSS responsivo e toda a estrutura do projeto. Enquanto a IA construía, eu respondia as questões diretamente no terminal e ia sendo corrigido em tempo real — uma sessão de estudo de verdade.
