Parte 2 — O que aparece no output?

Questão 6 — O que aparece no alert?

var x = 10; var y = "10"; alert(x == y); alert(x === y);

Primeiro alert: true

Segundo alert: false

== (igualdade normal) — compara apenas o valor, fazendo conversão automática de tipo. 10 == "10" é true porque o JavaScript converte a string "10" para número.

=== (igualdade estrita) — compara valor e tipo, sem conversão. 10 === "10" é false porque um é number e o outro é string.

Dica: na dúvida, use === para evitar surpresas.

Questão 7 — Quais valores possíveis para n?

var n = Math.floor(Math.random() * 4); alert(n);

Valores: 0, 1, 2 ou 3

Math.random() * 4 gera um decimal entre 0 e 3,999… O Math.floor() arredonda para baixo, resultando em 0, 1, 2 ou 3.

Regra geral: Math.floor(Math.random() * N) sorteia de 0 até N−1.

Questão 8 — O que aparece no alert?

var a = 7; var b = 3; var resultado = a % b; alert(resultado);

Resultado: 1 — 7 dividido por 3 dá 2, com resto de 1.

Questão 9 — Funciona do jeito esperado? Tem problema?

var nome = prompt("Nome:"); var idade = prompt("Idade:"); if (idade >= 18) { alert("Maior de idade"); } else { alert("Menor de idade"); }

Por coincidência funciona! O JavaScript faz coerção automática de tipo, convertendo a string para número na comparação com >=.

Problema: se o usuário digitar algo que não é número (ex: "abc"), o resultado será NaN e nenhuma condição faz sentido. O correto é usar parseFloat ou parseInt:

var idade = parseFloat(prompt("Idade:"));

Parte 3 — Escreva o código

Questão 10 — Média de 3 notas

Peça 3 notas do usuário, calcule a média e mostre:

  • Média >= 7 → "Aprovado"
  • Média >= 5 e < 7 → "Recuperação"
  • Média < 5 → "Reprovado"

var nota1 = parseFloat(prompt("Nota 1:")); var nota2 = parseFloat(prompt("Nota 2:")); var nota3 = parseFloat(prompt("Nota 3:")); var media = (nota1 + nota2 + nota3) / 3; if (media >= 7) { alert("Média: " + media.toFixed(2) + " — Aprovado!"); } else if (media >= 5) { alert("Média: " + media.toFixed(2) + " — Recuperação"); } else { alert("Média: " + media.toFixed(2) + " — Reprovado"); }
