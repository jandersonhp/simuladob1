Simulado 1 — Teoria, Output e Codigo

Parte 1 — Teoria (Origem do JS)

  1. Quem criou o JavaScript e em qual ano?
  2. Em quantos dias o JavaScript foi criado inicialmente?
  3. Qual era o nome original antes de virar "JavaScript"?
  4. JavaScript e Java sao a mesma coisa? Explique.
  5. Qual organizacao padroniza o JavaScript ate hoje (nome do padrao)?

1. Brendan Eich, em 1995, trabalhando na Netscape.

2. Aproximadamente 10 dias.

3. Primeiro se chamou Mocha, depois LiveScript. Nada a ver com cafe — "Java" era jogada de marketing da Sun (dona do Java) com a Netscape.

4. Nao! Sao linguagens completamente diferentes, sem relacao tecnica. Java e da Sun/Oracle, JavaScript e da Netscape/Mozilla. O nome similar foi estrategia de marketing.

5. A ECMA International padroniza o JavaScript. O padrao se chama ECMAScript. A Microsoft criou o JScript, que era um clone do JS no Internet Explorer — talvez seu professor tenha falado sobre isso quando disse que "a Microsoft roubou".

Parte 2 — O que aparece no output?

Questao 6 — O que aparece no alert?

var x = 10; var y = "10"; alert(x == y); alert(x === y);

Primeiro alert: true

Segundo alert: false

== compara valor (com conversao) → verdadeiro.

=== compara valor E tipo (sem conversao) → falso, pois number ≠ string.

Questao 7 — Quais valores possiveis para n?

var n = Math.floor(Math.random() * 4); alert(n);

Valores: 0, 1, 2 ou 3

Regra: Math.floor(Math.random() * N) sorteia de 0 ate N-1.

Questao 8 — O que aparece no alert?

var a = 7; var b = 3; var resultado = a % b; alert(resultado);

Resultado: 1 (resto de 7 ÷ 3 = 2 com resto 1).

Questao 9 — Funciona do jeito esperado?

var nome = prompt("Nome:"); var idade = prompt("Idade:"); if (idade >= 18) { alert("Maior de idade"); } else { alert("Menor de idade"); }

Por coincidencia funciona! O JS faz coercao automatica. Mas se o usuario digitar algo invalido vira NaN. O certo e usar parseFloat ou parseInt.

Parte 3 — Escreva o codigo

Questao 10 — Media de 3 notas

Pede 3 notas, calcula a media e mostra:

  • Media >= 7 → "Aprovado"
  • Media >= 5 e < 7 → "Recuperacao"
  • Media < 5 → "Reprovado"

var nota1 = parseFloat(prompt("Nota 1:")); var nota2 = parseFloat(prompt("Nota 2:")); var nota3 = parseFloat(prompt("Nota 3:")); var media = (nota1 + nota2 + nota3) / 3; if (media >= 7) { alert("Media: " + media.toFixed(2) + " — Aprovado!"); } else if (media >= 5) { alert("Media: " + media.toFixed(2) + " — Recuperacao"); } else { alert("Media: " + media.toFixed(2) + " — Reprovado"); }
