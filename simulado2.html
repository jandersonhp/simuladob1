Marque a alternativa e clique "Verificar" para ver se acertou

Questao 1

var x = "5"; var y = 3; alert(x + y);
  • a) 8
  • b) 53
  • c) 15
  • d) undefined
Correta: b) 53

Quando um dos operandos e string, o + concatena. "5" + 3 = "53".

Questao 2

Qual trecho sorteia um numero de 1 a 6?

var dado = _________ ; alert(dado);
  • a) Math.random() * 6
  • b) Math.floor(Math.random() * 6)
  • c) Math.floor(Math.random() * 6) + 1
  • d) Math.random(1, 6)
Correta: c) Math.floor(Math.random() * 6) + 1

Math.random() * 6 → 0..5.999, floor → 0..5, + 1 → 1..6.

Questao 3

var a = 3; var b = a * 2; var c = a + b; alert(c);
  • a) 3
  • b) 6
  • c) 9
  • d) 12
Correta: c) 9

b = 3 * 2 = 6. c = 3 + 6 = 9.

Questao 4

var nota = "7"; var passou = nota > 6; alert(passou);
  • a) true
  • b) false
  • c) undefined
  • d) Erro
Correta: a) true

O JS converte "7" para numero automaticamente na comparacao com >. Entao 7 > 6 = true.

Questao 5

Qual trecho testa se o numero esta entre 10 e 20?

var n = parseInt(prompt("Numero:")); if (______) { alert("Entre 10 e 20"); }
  • a) n > 10 e n < 20
  • b) n > 10 && n < 20
  • c) n > 10 || n < 20
  • d) 10 < n < 20
Correta: b) n > 10 && n < 20

Em JS, o operador E logico e &&. Opcao c) usa OU, d) nao funciona.

Questao 6

var x = 10; var y = 3; alert(x % y);
  • a) 3
  • b) 1
  • c) 3.33
  • d) 0
Correta: b) 1

10 ÷ 3 = 3, resto 1. O operador % retorna o resto da divisao inteira.

Questao 7

Qual a sequencia de outputs?

var idade = 17; if (idade >= 18) { alert("A"); } else { alert("B"); } alert("Fim");
  • a) Primeiro "B", depois "Fim"
  • b) So "Fim"
  • c) Primeiro "A", depois "Fim"
  • d) "B" (e para)
Correta: a) Primeiro "B", depois "Fim"

17 nao e maior que 18 → else → "B". Codigo fora do if SEMPRE executa → "Fim".

Questao 8

O que formata o IMC com 2 casas decimais?

var imc = peso / (altura * altura); alert("Seu IMC: " + __________ );
  • a) imc.toFixed(2)
  • b) imc.round(2)
  • c) Math.floor(imc)
  • d) imc.casas(2)
Correta: a) imc.toFixed(2)

O metodo .toFixed(n) formata com n casas decimais. Os outros nao existem.

Questao 9 (pegadinha)

var a = true; var b = false; var c = true; if (a && b || c) { alert("Sim"); } else { alert("Nao"); }
  • a) Sim
  • b) Nao
  • c) undefined
  • d) Erro
Correta: a) Sim

&& tem precedencia sobre ||. true && false = false. false || true = true → "Sim".

Questao 10 — Escreva o codigo e rode

Escreva um programa que:

  1. Sorteia um numero de 1 a 100
  2. Pede pro usuario digitar um palpite
  3. Se acertou → "Acertou!"
  4. Se palpite > sorteado → "Menor!"
  5. Se palpite < sorteado → "Maior!"

var sorteado = Math.floor(Math.random() * 100) + 1; var palpite = parseInt(prompt("Seu palpite (1-100):")); if (palpite == sorteado) { alert("Acertou!"); } else if (palpite > sorteado) { alert("Menor! O numero era " + sorteado); } else { alert("Maior! O numero era " + sorteado); }
