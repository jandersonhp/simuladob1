1. História do JavaScript

Linguagem leve para criar interações dinâmicas em páginas web, aplicativos, servidores e jogos.

  • Criada em 1995 por Brendan Eich na Netscape
  • Originalmente para validar formulários HTML
  • Suportada por quase todos os navegadores
  • Fundamental junto com HTML e CSS
📦

6. Atividades Práticas

6.1 — Verificação de valores (soma >= 20)

Somar dois números e alertar se o resultado é maior, menor ou igual a 20.

var Numero = parseInt(prompt("insira valor 1: ")); var Numero2 = parseInt(prompt("insira valor 2: ")); var resultado = (Numero + Numero2); if (resultado >= 20) { alert("O resultado é " + resultado + " é maior ou igual a 20"); } else { alert("O valor de " + resultado + " é menor que 20"); }

6.2 — Cálculo de IMC

Programa que calcula o Índice de Massa Corporal com base no peso e altura.

var nome = prompt("Digite seu nome"); var peso = parseFloat(prompt("Digite seu peso")); var altura = parseFloat(prompt("Digite sua altura")); var imc = peso / (altura * altura); if (imc < 18.5) { alert("Olá " + nome + ", seu peso está baixo. IMC: " + imc.toFixed(2)); } else if (imc < 24.99) { alert("Olá " + nome + ", seu peso está normal. IMC: " + imc.toFixed(2)); } else if (imc < 29.99) { alert("Olá " + nome + ", você está com sobrepeso. IMC: " + imc.toFixed(2)); } else { alert("Olá " + nome + ", você está com obesidade. IMC: " + imc.toFixed(2)); }

6.4 — Jogo de Par ou Ímpar

Usuário joga contra a máquina, que escolhe números aleatórios entre 0 e 10.

var nome = prompt("Digite seu nickname"); var primeirojog = parseFloat(prompt("Digite o primeiro número da jogada")); var segundojog = Math.floor(Math.random() * 10); alert(segundojog); var soma = primeirojog + segundojog; if (soma % 2 == 0) { alert("Deu PAR! " + nome + " ganhou Resultado: " + soma); } else { alert("Deu ÍMPAR! " + nome + " perdeu Resultado: " + soma); }
